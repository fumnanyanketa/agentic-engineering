# My progress: Agentic Engineering

**North-star project:** AtlasOS, a self-improving platform of cooperating AI agents that runs
knowledge work end to end. Every unit's Build adds one component of it. See [`atlas/`](atlas/)
for the company brief, architecture map, and component roadmap.

**The ritual:** read a unit (page by page), do its Build as the AtlasOS component for that unit,
commit it, tick the box. Pick one coding agent or model as your main tool; skim how the others do
the same. Commands and model ids move fast, so verify against current docs as you go.

**The course:** 11 units, in [`combined/`](combined/index.html). Work top to bottom.
